## Instalação

 1. Criar pasta "workspace" na sua máquina. Exemplo: C:\Users\%username%\Documents\workspace\
 2. Clone (download) do projeto framework (dependência): https://github.com/rogerfdeazevedo/rfaengines-testautomation-selenium
 3. Instalação/Atualização do Java: https://www.devmedia.com.br/preparacao-do-ambiente-para-desenvolvimento-em-java/25188
 4. Instalação/Atualização do Maven: http://luizricardo.org/2014/06/instalando-configurando-e-usando-o-maven-para-gerenciar-suas-dependencias-e-seus-projetos-java/
 5. Criar pasta dos drivers na unidade C:. Exemplo.: C:\selenium\drivers\%browser%\%versao%\
 6. Download drivers: https://www.seleniumhq.org/download/
